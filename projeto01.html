<Tag P>

Essa é uma tag de parágrafo, ela serve para separar blocos de texto.

<Tag I>

Essa é uma tag de itálico, ela serve para destacar palavras ou frases.

<Tag Strong>

Essa é uma tag de negrito, ela serve para enfatizar palavras ou frases.

<Tag U>

Essa é uma tag de sublinhado, ela serve para destacar palavras ou frases.

<Tag Mark>

Essa é uma tag de destaque, ela serve para marcar palavras ou frases importantes.

<Tag B>

Essa é uma tag de negrito, ela serve para enfatizar palavras ou frases, mas não é recomendada para uso semântico.

<Tag S>

Essa é uma tag de riscado, ela serve para indicar que algo foi removido ou não é mais relevante.

<Tag Small>

Essa é uma tag de texto pequeno, ela serve para diminuir o tamanho do texto.

<Tag Sub>

Essa é uma tag de subescrito, ela serve para colocar texto abaixo da linha base do texto normal. Exemplo: H2O

<Tag Sup>

Essa é uma tag de sobrescrito, ela serve para colocar texto acima da linha base do texto normal. Exemplo: E=mc2

<Tag Del>

Essa é uma tag de texto riscado, ela serve para indicar que algo foi removido ou não é mais relevante.

<Tag Ins>

Essa é uma tag de texto sublinhado, ela serve para indicar que algo foi adicionado ou é relevante.

<Tag Code>

Essa é uma tag de código, ela serve para exibir trechos de código ou comandos.

<Tag Pre> 

    Essa é uma tag de pré-formatação, ela serve para exibir texto com formatação preservada, como quebras de linha e espaços.

<Tag Blockquote>

Essa é uma tag de citação em bloco, ela serve para destacar citações longas ou importantes.

— Autor da citação
<Tag Q>

Essa é uma tag de citação curta, ela serve para destacar citações curtas ou trechos de texto. Exemplo: Essa é uma citação curta.

<Tag Abbr>

Essa é uma tag de abreviação, ela serve para indicar que uma palavra ou frase é uma abreviação. Exemplo: HTML é a linguagem de marcação usada para criar páginas web.

<Tag Address>

Essa é uma tag de endereço, ela serve para exibir informações de contato ou localização. Exemplo:

Endereço: Rua Exemplo, 123, Cidade, Estado

<Tag Time>

Essa é uma tag de tempo, ela serve para exibir datas e horários. Exemplo: 1 de Outubro de 2023

<Tag Cite>

Essa é uma tag de citação, ela serve para indicar a fonte de uma citação ou referência. Exemplo: Nome do Autor escreveu um livro chamado O Livro Exemplo.

<Tag Em>

Essa é uma tag de ênfase, ela serve para destacar palavras ou frases. Exemplo: Essa é uma frase enfatizada.

<Tag Dfn>

Essa é uma tag de definição, ela serve para definir termos ou conceitos. Exemplo: Definição é uma explicação de um termo ou conceito.

<Tag Kbd>

Essa é uma tag de teclado, ela serve para exibir atalhos de teclado. Exemplo: Ctrl + C é o atalho para copiar.

<Tag Var>

Essa é uma tag de variável, ela serve para exibir variáveis em programação ou matemática. Exemplo: x é uma variável que pode representar qualquer valor.

<Tag Samp>

Essa é uma tag de amostra, ela serve para exibir saídas de programas ou exemplos de código. Exemplo: Exemplo de saída de um programa é o resultado de uma execução.

<Tag Br>

Essa é uma tag de quebra de linha, ela serve para inserir quebras de linha no texto. Exemplo: Esta é uma linha.
Esta é outra linha.

Essa é uma tag de direção de texto, ela serve para alterar a direção do texto. Exemplo: Texto em direção da direita para a esquerda é usado para exibir texto em idiomas que são lidos da direita para a esquerda.

<Tag Span>

Essa é uma tag de contêiner genérico, ela serve para aplicar estilos ou scripts a partes do texto. Exemplo: Texto em vermelho é usado para aplicar estilos específicos a partes do texto.

<Tag Table>

Essa é uma tag de tabela, ela serve para exibir dados em formato tabular. Exemplo:

Tabela de Exemplo
Nome Idade Cidade
Alice 30 São Paulo
Bob 25 Rio de Janeiro
Charlie 35 Belo Horizonte
Total de pessoas: 3

Você pode usar a tag table para exibir dados em formato tabular. A tag table é composta por várias outras tags, como caption, thead, tbody, tr, td e th, que são usadas para estruturar a tabela.


<Tag Caption>

Essa é uma tag de legenda de tabela, ela serve para fornecer um título ou descrição para uma tabela. Exemplo:

Tabela de Exemplo
Nome Idade Cidade
Alice 30 São Paulo
Bob 25 Rio de Janeiro
Charlie 35 Belo Horizonte
Total de pessoas: 3

Você pode usar a tag caption para fornecer um título ou descrição para uma tabela. A tag caption deve ser usada dentro da tag table e antes das tags thead, tbody ou tfoot.


<Tag Thead>

Essa é uma tag de cabeçalho de tabela, ela serve para agrupar as linhas de cabeçalho de uma tabela. Exemplo:

Tabela de Exemplo
Nome Idade Cidade
Alice 30 São Paulo
Bob 25 Rio de Janeiro
Charlie 35 Belo Horizonte
Total de pessoas: 3

Você pode usar a tag thead para agrupar as linhas de cabeçalho de uma tabela. A tag thead deve ser usada dentro da tag table e antes das tags tbody ou tfoot. Ela ajuda a identificar as colunas da tabela e melhora a acessibilidade.


<Tag Tbody>

Essa é uma tag de corpo de tabela, ela serve para agrupar as linhas de dados de uma tabela. Exemplo:

Tabela de Exemplo
Nome Idade Cidade
Alice 30 São Paulo
Bob 25 Rio de Janeiro
Charlie 35 Belo Horizonte
Total de pessoas: 3

Você pode usar a tag tbody para agrupar as linhas de dados de uma tabela. A tag tbody deve ser usada dentro da tag table e depois da tag thead. Ela ajuda a organizar os dados da tabela e melhora a acessibilidade.


<Tag Tr>

Essa é uma tag de linha de tabela, ela serve para definir uma linha dentro de uma tabela. Exemplo:

Tabela de Exemplo
Nome Idade Cidade
Alice 30 São Paulo
Bob 25 Rio de Janeiro
Charlie 35 Belo Horizonte
Total de pessoas: 3

Você pode usar a tag tr para definir uma linha dentro de uma tabela. A tag tr deve ser usada dentro das tags thead, tbody ou tfoot. Ela agrupa as células de uma linha e ajuda a organizar os dados da tabela.


<Tag Td>

Essa é uma tag de célula de tabela, ela serve para definir uma célula dentro de uma linha de tabela. Exemplo:

Tabela de Exemplo
Nome Idade Cidade
Alice 30 São Paulo
Bob 25 Rio de Janeiro
Charlie 35 Belo Horizonte
Total de pessoas: 3

Você pode usar a tag td para definir uma célula dentro de uma linha de tabela. A tag td deve ser usada dentro da tag tr. Ela contém os dados de uma célula e ajuda a organizar os dados da tabela.


<Tag Th>

Essa é uma tag de cabeçalho de célula, ela serve para definir uma célula de cabeçalho dentro de uma linha de tabela. Exemplo:

Tabela de Exemplo
Nome Idade Cidade
Alice 30 São Paulo
Bob 25 Rio de Janeiro
Charlie 35 Belo Horizonte
Total de pessoas: 3

Você pode usar a tag th para definir uma célula de cabeçalho dentro de uma linha de tabela. A tag th deve ser usada dentro da tag tr. Ela contém o título ou cabeçalho de uma coluna e ajuda a identificar os dados da tabela.


<tag tfoot

Essa é uma tag de rodapé de tabela, ela serve para agrupar as linhas de rodapé de uma tabela. Exemplo:

Tabela de Exemplo
Nome Idade Cidade
Alice 30 São Paulo
Bob 25 Rio de Janeiro
Charlie 35 Belo Horizonte
Total de pessoas: 3

Você pode usar a tag tfoot para agrupar as linhas de rodapé de uma tabela. A tag tfoot deve ser usada dentro da tag table e depois das tags thead e tbody. Ela contém informações adicionais sobre a tabela, como totais ou resumos.
